Governing Thought

泼水节赛单是 36.2 版本表現最佳的限時赛单：日均參與玩家（71,562）是同期 Event 版本赛单均值的 3.7 倍，中退率（4.01%）僅為其 1/4，且在無 IP 加持的條件下達到 Core 常態赛单水準——投入值得，且驗證了「節日主題本地化赛单」作為可複製路徑的可行性。

Supporting Argument 1：泼水节參與規模全面碾壓同期 Event 版本赛单

Fact：泼水节日均參與玩家 71,562 人，是同期 6 個 Event 版本赛单均值（19,364）的 3.7 倍；玩家參與率 3.23%，是 Event 均值（0.98%）的 3.3 倍。第二名 power up（32,613）僅為其 46%。14 天累計 1,001,865 人次·天、3,187,247 局次。

SA圖｜同版本並行赛单對比（日均玩家 × 中退率）

泼水节不僅參與量是第二名的 2.2 倍，且中退率僅 4.0%——唯一同時在「規模」和「品質」兩個維度領先的赛单。Carnival Shootout 參與量低且中退率 52.8%，反映體驗設計失敗。

Implication：泼水节的參與規模不是小幅領先而是量級差距。「節日主題 × 本地化」的內容定位具有獨立的拉動力，遠超版本自帶的 Event 赛单。

Action：後續版本規劃時，將本地化節日赛单的優先級提升至與 IP 聯動赛单同級——數據顯示其不依賴 IP 即可達到顯著的參與拉動。

Supporting Argument 2：中退率 4.01% 證明遊戲體驗本身具有吸引力

Fact：泼水节中退率 4.01%，排名第二低（僅次於 Enhanced Warrior 2.41%），遠低於 Event 版本赛单均值 15.89%、Core 常態均值 9.49%。搭配人均 3.18 局（Event 均值 2.42 的 1.31x、Core 均值 2.92 的 1.09x），玩家不僅玩完每局還主動重複遊玩。

中退率排名（同期所有赛单 + 基準線）

泼水节中退率低於 Core 常態均值（9.49%），說明其體驗品質不僅是「限時赛单中的優秀」，而是「優於玩家日常遊玩的基礎模式」。

Implication：低中退率 = 玩家不願中途退出 = 遊戲體驗令人滿意。這是純玩法品質的驗證，與 IP 吸引力無關。

Action：提煉泼水节核心機制（固定水枪武器 + 增強滑鏟 + 快速技能充能 + 短局 2.9 分鐘）作為後續本地化赛单設計範本。

Supporting Argument 3：4/1 飙升 +74% 驗證文化節點叠加效應

Fact：泼水节上線 14 天參與趨勢平穩（日均 63K-76K），4/1 突然飙升至 118,237 人（+74% vs 前日），參與率 3.13% → 4.81%，局次占比 9.40% → 12.23%。4/1 正值 TH 暑假開始 + 泼水節日期臨近（4/13-15）。

14 天日參與趨勢

14 天趨勢平穩無衰退，4/1 的文化節點叠加帶來 +74% 爆發——如果延長至 4/13 實際泼水節，峰值可能更高。

Implication：文化主題赛单的價值在於接近節日時的爆發力。4/1 的飙升可能只是前奏而非峰值。這是「內容與文化時機匹配」的直接證據。

Action：（1）延長赛单至 4/15 覆蓋實際泼水節。（2）後續本地化赛单應提前節日 2-3 週上線，讓文化認同逐步發酵。

Supporting Argument 4：無 IP 加持下達到 Core 常態水準，驗證玩法驅動的可行性

Fact：泼水节在無 IP 授權條件下——日均參與 71,562 = Core 常態均值 65,616 的 1.09x；人次參與占比 9.21% ≈ Core 均值 9.87%；中退率 4.01% 遠優於 Core 均值 9.49%；人均局數 3.18 > Core 均值 2.92。與歷史 IP 聯動比較：達到 Gundam 峰值（275,548）的 26%、超越 WWE（45,661）的 1.57x、超越装卸货场-新春（36,203）的 1.98x。
