Governing Thought

ALL C7 27.4% 距警報線 2.3pp 只是表面——拆分市場後發現 5 個非 PH 市場的 C7 已全線超過 30%（每天），被 PH 的量體掩蓋。最危險的是 ID（+2.9pp，月末加速至 40.2%）和 MY（+3.0pp），兩者均與 Ramadan/Lebaran 期間行為結構性變化有關但仍需觀察 4 月是否回落。BR 偏好玩家的 D1 留存降幅最大（-2.1pp），加上 A+B 池全線萎縮，形成「核心玩家降池 → 匹配品質下降 → 中層加速流出」的負循環。4 月需分市場啟動流失干預，而非僅依賴匯總警報線。

Supporting Argument 1：ALL C7 被 PH 量體掩蓋——5 個市場已全線超過警報線

Fact：ALL C7 27.4% 距警報線 29.7% 尚有 2.3pp 緩衝。但拆分市場：PH C7 ~26.5%（安全），TWHKMO 37.8%、TH 36.6%、ID 37.2%、SG 34.5%、MY 35.0%——5 個市場 3 月每天的 C7 都超過 30%。PH DAU 佔 85%，將 ALL C7 從真實的加權均值拉低了約 7-8pp。

Implication：如果只監控 ALL C7 警報線（29.7%），在 PH 穩定的情況下，其他 5 個市場的流失危機永遠不會觸發全局警報——即使這些市場已經處於「C7 > 35% 的嚴重流失」狀態。現有的單一全局警報線設計存在結構性盲點。

Action：⛔ 建議立即調整監控機制：(1) 新增各市場獨立 C7 警報線（基於各市場歷史基準值的 mean + 2σ）；(2) 觸發規則改為「任一市場 C7 超過該市場警報線」即觸發異動分析，不再依賴 ALL 匯總。Owner：活躍組 + 數據，Timeline：4 月第 1 週完成警報線設定。

Supporting Argument 2：ID C7 月末加速至 40.2% 是最緊急的單一市場風險

Fact：ID 3 月 C7 均值 37.2%（MoM +2.9pp），月內持續加速——W1 均值 ~35%、W3 ~37%、W5 39.8%，3/31 峰值 40.2%。C14 同步惡化。ID 是 3 月唯一 DAU MoM 正成長的市場（+6.2%），但其活躍 R7 僅 52.8%（基準 57.7%，-4.9pp），為所有市場最大降幅。3 月正值 Ramadan 結束 + Lebaran 假期（3/21-24）+ Mudik 返鄉潮。

Implication：ID 的 DAU 正成長和 C7 加速是矛盾但可解釋的：Lebaran 後大量回流用戶（Revival +9.1%）推高了 DAU，但這些用戶是低品質回流（R7 拉低至 52.8%），7 天內大量再流失推高了 C7。Mudik 返鄉潮期間的行為數據可能失真（設備切換、網路劣化）。關鍵問題：4 月 Ramadan 完全結束後，C7 是否會自然回落？如果不回落，代表不只是假期效應。

Action：4 月第 1 週觀察 ID C7 是否回落至 35% 以下。如果 4/7 前仍 > 37%，確認非假期效應，需啟動 ID 專項流失分析。同步追蹤 3 月 Lebaran 回流用戶的 R7/R14——如果這批回流的 R14 < 10%，代表回流品質極差，需重新評估 ID 的回流策略。Owner：活躍組 + 數據，Timeline：4/7 為判斷節點。

Supporting Argument 3：BR 偏好玩家流失最快——競品 IP 聯名的直接影響

Fact：跨市場 BR 偏好玩家 D1 留存降幅最大：ALL -2.1pp（76.6% → 74.5%），PH -2.1pp，ID -2.3pp。BR 玩家佔比從 36.0% 降至 33.5%（-2.5pp），代表 BR 玩家正在離開。同期 PUBG Mobile JJK 聯名（3/13 起）同為 BR 品類直接競爭。MP 留存相對穩定（ALL -0.6pp），但 TWHKMO 的 MP D1 崩潰 -11.3pp 是例外。

Implication：BR 玩家的流失與 PUBG JJK 的時間高度吻合。BR 群是對 IP 聯名最敏感的族群——同品類的高品質 IP 活動會直接拉走 BR 核心玩家。C7 上升中的 BR 貢獻度可能高於其佔比——BR 玩家通常是 A/B 池的主力（A 池 BR% 68.0%），他們的流失同時壓低了 A 池人數和整體匹配品質。

Action：設計針對 BR 玩家的留池機制：(1) 4 月排位賽季獎勵加強（段位獎勵升級，避免 BR 玩家轉場 PUBG）；(2) 追蹤 BR 偏好 × A 池交叉的人數趨勢，確認 BR 流失是否是 A 池萎縮的主要驅動力。Owner：活躍組，Timeline：4 月第 1 週。

Supporting Argument 4：A+B 池萎縮 → C7 上升的因果鏈——負循環已啟動

Fact：A 池 MoM -19.2%（699K → 565K），B 池 -11.1%（591K → 526K）。D 池持平（1,837K），E+F+G 佔比從 47.1% 升至 48.5%。A 池日均遊戲 119 分鐘、日均 12.8 局，是維持匹配品質的核心。A 池 BR 偏好 68%——A 池流失 = BR 匹配池縮小。

Implication：因果鏈已形成：A/B 池玩家流失 → BR/MP 匹配池縮小 → 匹配等待時長增加 + Bot 比例上升 → 中層玩家（C/D 池）體驗劣化 → C/D 池加速流向 E/F/G → C7 上升。這是一個自我強化的負循環——人越少、體驗越差、走得越快。D 池持平但未升池代表升池動力已不足以對抗降池壓力。

Action：打斷負循環的關鍵是穩住 A/B 池人數。月報已建議的 7 天連續登入遞進獎勵需加速上線（原定 4 月第 2 週，建議提前至第 1 週）。同時評估匹配演算法是否可針對低人口市場優化（如 TWHKMO/TH 的跨池匹配）。Owner：活躍組 + 產品，Timeline：4 月第 1 週。

建議：調整 C7 監控機制

現有全局警報線（ALL C7 > 29.7%）的設計有結構性盲點。建議調整為：

層級當前機制建議機制
全局ALL C7 > 29.7% 觸發警報保留，但降為輔助指標
市場級（新增）無各市場獨立警報線（基於該市場 mean + 2σ）。任一市場觸發即啟動異動分析
分池級（新增）無追蹤 C/D 池的 C7——這是負循環的樞紐層。C/D 池 C7 上升代表降池壓力正在傳導
市場級警報線的具體數值需由數據團隊基於各市場歷史 C7 計算 mean ± 2σ。初步估算：PH ~29%、TWHKMO ~42%、TH ~39%、ID ~40%、SG ~37%、MY ~38%。
