赛单分類與對標基準

分類方法論

CODM 赛单數據依兩層邏輯分類。第一層由數據源原生欄位 playlist类型 區分 MP/BR/排位/錦標賽；第二層在「普通匹配」內部，按 playlistid 區間 + 描述名命名模式進一步細分。泼水节（500191）屬於「普通匹配 → 500xxx 限時赛单 → 節日/本地化」。

分析範圍限定為 MP 普通匹配——泼水节是 MP 限時玩法，與 BR 模式的參與結構和指標意義完全不同，不適合直接對標。

Governing Thought

泼水节赛单是 36.2 版本表現最佳的限時赛单：日均參與玩家（71,562）為同期 Event 版本赛单均值的 3.7 倍，中退率（4.01%）僅為其 1/4，且在無 IP 授權的條件下達到 Core 常態赛单水準。數據顯示「節日主題本地化赛单」作為可複製的內容路徑具備可行性，建議在後續版本中延續。

Supporting Argument 1：泼水节日均參與達同期 Event 赛单均值的 3.7 倍

Fact：泼水节日均參與玩家 71,562 人，是同期 6 個 Event 版本赛单均值（19,364）的 3.7 倍；玩家參與率 3.23%，是 Event 均值（0.98%）的 3.3 倍。第二名 power up（32,613）僅為其 46%。14 天累計 1,001,865 人次·天、3,187,247 局次。

SA圖｜同版本並行赛单對比（日均玩家 × 中退率）

泼水节不僅參與量是第二名的 2.2 倍，且中退率僅 4.0%——唯一同時在「規模」和「品質」兩個維度領先的赛单。Carnival Shootout 參與量低且中退率 52.8%，反映體驗設計失敗。

Implication：泼水节的參與規模與同期 Event 赛单存在量級差距（3.7x），顯示「節日主題 × 本地化」的內容定位具有獨立的參與拉動力，不依賴 IP 授權即可達成。

Action：建議後續版本規劃中，將本地化節日赛单的優先級提升至與 IP 聯動赛单同級，作為常態化的內容品類納入版本排期。

Supporting Argument 2：中退率 4.01%，低於 Event 均值 15.89% 與 Core 均值 9.49%

Fact：泼水节中退率 4.01%，排名第二低（僅次於 Enhanced Warrior 2.41%），遠低於 Event 版本赛单均值 15.89%、Core 常態均值 9.49%。搭配人均 3.18 局（Event 均值 2.42 的 1.31x、Core 均值 2.92 的 1.09x），玩家不僅玩完每局還主動重複遊玩。

中退率排名（同期所有赛单 + 基準線）

泼水节中退率低於 Core 常態均值（9.49%），說明其體驗品質不僅是「限時赛单中的優秀」，而是「優於玩家日常遊玩的基礎模式」。

Implication：低中退率 = 玩家不願中途退出 = 遊戲體驗令人滿意。這是純玩法品質的驗證，與 IP 吸引力無關。

Action：建議將泼水节的核心機制設計（固定趣味武器、增強滑鏟距離、快速技能充能）提煉為本地化赛单的設計範本，供後續版本復用。關鍵公式：短局（<3 分鐘）+ 低門檻 + 節日主題機制融合。

Supporting Argument 3：4/1 參與飆升 +74%，與 PH 暑假啟動時間吻合

Fact：泼水节上線 14 天參與趨勢平穩（日均 63K-76K），4/1 突然飆升至 118,237 人（+74% vs 前日），參與率 3.13% → 4.81%，局次占比 9.40% → 12.23%。4/1 正值 PH 暑假啟動（約 3/28 起），PH 佔全區 DAU 約 85%，ALL 數據的飆升主要反映 PH 市場的暑假效應。TH 泼水節文化認同（4/13-15 臨近）可能為次要疊加因素，但因無分市場數據，其獨立貢獻度無法量化。

14 天日參與趨勢

14 天趨勢平穩無衰退，4/1 與 PH 暑假啟動吻合帶來 +74% 飆升——若延長至 4/13 實際泼水節，TH 市場可能帶來額外峰值。

Implication：4/1 飆升的主要驅動力為 PH 暑假啟動帶來的整體 DAU 增長，泼水节赛单作為當期最具吸引力的限時內容，自然承接了增量用戶。同時，泼水節文化主題可能對 TH 市場有額外拉動，若延長至實際泼水節（4/13-15），有機會進一步驗證文化節點效應。

Action：（1）建議延長泼水节赛单至 4/15（覆蓋實際泼水節），測試文化節點高峰期的參與潛力。（2）後續本地化赛单的上線時間建議提前於節日 2-3 週，讓主題與節日氛圍同步升溫。

Supporting Argument 4：無 IP 授權下達到 Core 常態水準，玩法本身具備參與拉動力

Fact：泼水节在無 IP 授權條件下——日均參與 71,562 = Core 常態均值 65,616 的 1.09x；人次參與占比 9.21% ≈ Core 均值 9.87%；中退率 4.01% 遠優於 Core 均值 9.49%；人均局數 3.18 > Core 均值 2.92。與歷史 IP 聯動比較：達到 Gundam 峰值（275,548）的 26%、超越 WWE（45,661）的 1.57x、超越装卸货场-新春（36,203）的 1.98x。
