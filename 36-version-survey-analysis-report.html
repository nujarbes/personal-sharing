4. 分析脈絡與交叉分析設計

以下三條主軸貫穿整份分析，每條軸線對應不同的業務問題。

軸線一：版本內容是否真的影響留存？

核心假說：高版本滿意度 → 更高留存意願 → 更高遊玩頻率

Q19（版本滿意度 1-5）× Q20（版本留存意願）× Q6（遊玩頻率）
分群維度：玩家類型（新進 / 活躍 / 回流）
預期輸出：2x3 矩陣（版本滿意高/低 × 頻率高/中/低），找出「版本滿意度高但頻率下降」的異常群體（可能有其他推力）；以及「版本滿意度低但頻率不變」的群體（核心玩家黏性足夠）
版本滿意 Top-box版本滿意 Bottom-box
頻率高（每週5-7天）✅ 健康狀態⚠️ 黏性強但對版本不滿 → 找版本問題點
頻率低（每週0-4天）⚠️ 喜歡版本但不玩 → 找外部推力🚨 高流失風險 → 最需深入分析

軸線二：玩家流失是推力還是競品拉力？

核心假說：流失原因可分為「競品吸走」vs「CODM 自身問題」兩類，需分開應對

Q15（流失主因）× Q8（競品遊玩狀況）× Q10（競品吸引力要素）
分析方法：
Step 1：比較「有競品重疊玩家」vs「無競品重疊玩家」的 Q15 流失原因分布差異
Step 2：Q10 競品吸引力高頻要素 = CODM 相對弱點，與 Q15 流失原因對齊找重疊點
Step 3：流失原因中屬於「CODM 可以改善」的（如：武器平衡、匹配機制）vs「競品提供差異化」的（如：美術風格），優先解決可控問題

軸線三：商業化轉化阻礙在哪裡？

核心假說：「知道但沒買」的玩家中存在可轉化空間，需識別關鍵阻礙

Q29（轉盤參與狀況）→ 「很想買但沒下手」者 → Q31（未購買原因）
分群維度：年齡、頻率
預期輸出：每個轉盤的漏斗（認知率 → 有購買意願比率 → 實際購買率），「認知率高 + 意願率高 + 購買率低」的轉盤 = 設計或定價問題最集中的地方
轉盤認知率有購買意願實際購買率分析重點
